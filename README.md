# sistema-carneiro-ramalho
Descrição para GitHub Este é um projeto para um Sistema de Gestão de Vagas e Currículos, desenvolvido para conectar empresas e candidatos. A plataforma possui três níveis de acesso distintos: Administrador, Empresa e Candidato, cada um com funcionalidades e painéis exclusivos para gerenciar seus respectivos perfis e atividades.
