Vagas Online - Sistema de Gestão de Vagas e Currículos
Sobre o Projeto
O Vagas Online é uma plataforma robusta e escalável desenvolvida para conectar empresas e profissionais de maneira eficiente. O sistema oferece três níveis de acesso distintos — Administrador, Empresa e Candidato — cada um com painéis e funcionalidades adaptados às suas necessidades específicas.

O projeto foi construído com uma stack de desenvolvimento web clássica e poderosa, garantindo alto desempenho e facilidade de manutenção.

Funcionalidades Principais
Para Candidatos
Gerenciamento de Perfil: Crie e edite seu perfil profissional, adicione experiências, formação acadêmica e áreas de interesse.

Upload de Currículo: Faça o upload do seu currículo em formato PDF.

Busca de Vagas: Encontre oportunidades de emprego usando filtros avançados (área, cidade, tipo de contrato, faixa salarial).

Acompanhamento de Candidaturas: Monitore o status das suas inscrições em vagas (Inscrito, Em Avaliação, Aprovado, Rejeitado).

Para Empresas
Controle de Conta: Aguarde a aprovação do Administrador para ativar sua conta e publicar vagas.

Criação de Vagas: Publique novas vagas com detalhes completos (título, descrição, requisitos, salário, etc.).

Pipeline de Candidatos: Gerencie o processo seletivo com um painel visual, movendo candidatos entre as etapas do funil (inscritos, em avaliação, entrevistados, aprovados).

Para Administradores
Moderação de Conteúdo: Controle a aprovação de novas empresas e vagas antes que elas se tornem públicas.

Gerenciamento de Usuários: Visualize, edite e bloqueie contas de usuários e empresas.

Relatórios e Métricas: Acesse um painel completo com estatísticas sobre o uso da plataforma (número de usuários, vagas ativas, candidaturas, etc.).

Tecnologias Utilizadas
Backend: Wagtail/jango

Frontend: HTML5, CSS, JavaScript

Banco de Dados: SQLite/ PostgreSQL
